# Anthie Landing

Este es el repositorio para la landing page de **Anthie**, construida con [Astro](https://astro.build/) y [Tailwind CSS](https://tailwindcss.com/).

## 🚀 Inicio Rápido

Para instalar las dependencias y ejecutar el proyecto localmente, utiliza los siguientes comandos:

```bash
# Instalar dependencias
npm install

# Iniciar el servidor de desarrollo en http://localhost:4321
npm run dev
```

## 🧞 Comandos Útiles

| Comando | Acción |
| :--- | :--- |
| `npm install` | Instala las dependencias del proyecto. |
| `npm run dev` | Inicia el servidor de desarrollo local. |
| `npm run build` | Compila el sitio para producción en la carpeta `dist/`. |
| `npm run preview` | Previsualiza la compilación de producción localmente. |
| `npm run astro ...` | Ejecuta comandos CLI de Astro (ej. `astro add`, `astro check`). |

## 📁 Estructura del Proyecto

Dentro del proyecto de Astro, verás la siguiente estructura de archivos y carpetas:

```text
/
├── public/
│   ├── robots.txt
│   └── ...
├── src/
│   ├── components/
│   ├── layouts/
│   └── pages/
├── astro.config.mjs
├── package.json
└── tailwind.config.mjs
```

Astro utiliza un enrutamiento basado en archivos. Cada archivo con la extensión `.astro` dentro de la carpeta `src/pages/` se convertirá en una ruta en tu sitio web.
